# Command Line Interface

After installation, the following console scripts are available:

```bash
cm-make-figures
cm-opcode-test
cm-opcode16-test
cm-compound-opcode-test
cm-serial-adder-test
cm-kbit-memory-test
cm-kxor-visual
cm-rule110-gif
```

Repository-level experiments are run directly:

```bash
python -m experiments.run_toy_addition_experiment
python -m experiments.compare_readouts
```

Example:

```bash
cm-make-figures
```
